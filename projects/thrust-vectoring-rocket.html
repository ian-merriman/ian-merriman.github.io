Thrust-Vectoring Rocket Project

An early-stage rocket control project using IMU sensor data and servo actuation to explore thrust-vectoring concepts, feedback control, and vehicle attitude response.

Controls IMU Integration Embedded Systems Servo Actuation

Project Overview

This project was created to practice the fundamentals of thrust vector control on a small hardware testbed. The goal was to connect an inertial measurement unit to a microcontroller and use orientation data to command a servo-based mechanism that could represent thrust vectoring behavior.

My Role

I worked on the early concept, hardware setup, control logic, and documentation for the project. My focus was on understanding how sensor data can be converted into actuator commands and how feedback control concepts apply to rocket attitude control.

Technical Approach

Inputs

  • IMU orientation and angular rate data
  • Microcontroller input/output signals
  • Servo command limits
  • Desired attitude or neutral position

Outputs

  • Servo deflection commands
  • Basic sensor-to-actuator response
  • Initial feedback control behavior
  • Lessons for future TVC testbed development

What I Learned

This project helped me better understand how guidance and control concepts connect to physical hardware. It also showed the importance of filtering noisy sensor data, respecting actuator limits, and building simple test cases before moving toward more advanced control laws.

Future Improvements

Future work could include implementing a complementary filter or Kalman filter, adding PID control, logging sensor data, improving the mechanical TVC mount, and developing a safer bench-test procedure before any flight-related testing.
